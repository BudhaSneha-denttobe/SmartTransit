# 🚌 SmartTransit

### AI-Powered Smart Transportation Platform

A modern smart transportation system that helps commuters track buses in real time, plan routes efficiently, receive weather-aware travel recommendations, and access transit information through an AI-powered assistant.

---
Live Link:https://smart-transit-amvc.vercel.app/
# ✨ Key Features

### 🚍 Live Bus Tracking

* Track buses in real time on an interactive map.
* View current bus locations and movement updates.
* Monitor routes with dynamic position updates.

### 🗺️ Smart Route Planner

* Find the best route between any two bus stops.
* Get optimized travel suggestions.
* Reduce travel time with intelligent route recommendations.

### 🏢 Bus Stops & Route Information

* Browse all available bus stops and routes.
* View schedules and route details.
* Search and filter routes easily.

### 🤖 AI Transit Assistant

* Gemini-powered chatbot for transportation queries.
* Get instant answers about routes, buses, stops, and schedules.
* Provides personalized travel guidance.

### 🌦️ Weather-Aware Travel Recommendations

* Real-time weather integration.
* Travel suggestions based on current weather conditions.
* Alerts for rain, storms, and unfavorable travel situations.

### 🚨 Emergency SOS System

* One-click emergency alerts.
* Share live location instantly during emergencies.
* Designed for passenger safety and quick assistance.

### 👤 User Dashboard

* Secure user authentication and authorization.
* Manage profile information.
* Save favorite routes and bus stops.
* Access travel and search history.

### 📊 Admin Dashboard

* Manage buses, routes, and bus stops.
* Monitor platform activity and analytics.
* Perform complete CRUD operations.
* Centralized transit management system.
